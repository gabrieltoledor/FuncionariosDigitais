# Sprint 01

Objetivo:

Criar o primeiro Funcionário Digital capaz de atender uma empresa pelo WhatsApp.

## Funcionalidades

- Receber mensagens
- Entender perguntas
- Consultar a base de conhecimento
- Responder naturalmente
- Encaminhar para humano quando necessário

## Fora desta Sprint

- Dashboard
- Cobrança
- Multiempresa
- Agendamentos
- Recuperação automática
- Campanhas

## Meta

Ter um MVP funcionando no nosso próprio WhatsApp.
